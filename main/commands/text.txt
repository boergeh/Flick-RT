
flick text <sub_command> [argument ...] <file_name>

Usage 1:

  flick text xy file_with_tabulated_xy_data.txt
  
  List xy-values in a text file with two columns with xy-data. Any
  header text will be omitted.

Usage 2:

  flick text xy-precision <nx> <ny> file_with_tabulated_xy_data.txt

  List xy-values with a given precision, where 'nx' and 'ny' are the
  number of significant figures in x and y columns,
  respectively. Header text will be included.

  Example:

  flick text xy-precision 4 4 file_with_tabulated_xy_data.txt

Usage 3:

  flick text xy-scale <kx> <ky> file_with_tabulated_xy_data.txt

  List scaled xy-values, where 'kx' and 'ky' are the scaling factors
  for x and y columns, respectively. Header text will be included.

  Example:

  flick text xy-scale 1e-9 1e-4 file_with_tabulated_xy_data.txt
